# MusicPlayer
<p align="justify">
    A simple musicplayer built on top of javafx framework jdk21. 
    The application uses controlsfx and bootstrapfx. 
    The application is very simple to use.
    The application can be used to load a single audio file or an entire directory. 
    Optionally playlists can be made for easy access in the future.
</p>

NOT READY TO BE USED YET!!!

## Getting Started
### Dependencies
Java21

### Installation
Download the application from releases.

Open the `MusicPlayer.jar` file, in case double clicking does not work you may need to run the following command on the terminal.
```
java -jar ./MusicPlayer.jar
```
