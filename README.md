# Music Player
![Version](https://img.shields.io/badge/java-21+-blue.svg?style=flat)
![License](https://img.shields.io/badge/license-MIT-green.svg?style=flat)

<p align="justify">
    A simple musicplayer built with java21 on top of javafx framework. The application is very simple to use and has a very similar interface to some well known options. A lot of inspiration has been taken from the said options. The application contains an option to create persistent playlists and to play them at will. The application also contains a few different selector for playing a playlist in order or choosing a song at random. There is also an option to play songs that are not in any playlist, just open a file/directory and play the song(s).
</p>

- BootstrapFX

## Getting Started
### Installation for Users
1. Make sure to have Java installed.
2. Download the application from releases (NO RELEASES YET!).
3. Unzip the downloaded directory to a location of your choosing. The directory should contain `MusicPlayer-v1.0.0.jar` file (and some other files, license etc.), where the 1.0.0 is the version number and may wary.
4. Run the `MusicPlayer-v1.0.0.jar` file by double-clicking, in case the double-click does not work run one of the commands below depending on your operating system.

> [!NOTE]
> For commands to work you need to be inside the directory in which the jar file lives in!

**For Linux**
```
java -jar ./MusicPlayer.jar
```

**For Windows 10 and Windows 11**
```
java -jar .\MusicPlayer.jar
```

### Installation for Developers
1. Make sure to have git installed.
2. Clone the repository.
3. Run the application with the shell script (SCRIPT NOT BUILT YET!). For linux you may need to allow execution of the script.

**For Linux**
```
git clone https://github.com/WilliwadelmaWisky/MusicPlayer.git
```

```
chmod u+x ./MusicPlayer/MusicPlayer.sh
```

```
./MusicPlayer/MusicPlayer.sh
```

**For Windows**
```
git clone https://github.com/WilliwadelmaWisky/MusicPlayer.git
```

```
.\MusicPlayer\MusicPlayer.bat
```
